# Fase 0 - Preparacao e Alinhamento

Este pacote define o contrato de MVP para o projeto `mechanic-erp-agentic`.
Ele delimita escopo, fluxo operacional, dados, regras, integracao, padroes de execucao e gate de aceite.

## Definicao de pronto da Fase 0

A Fase 0 e considerada concluida quando:

- O escopo do MVP esta delimitado com inclui/exclui e fluxo minimo operacional.
- As jornadas por persona e excecoes comuns estao descritas.
- O dicionario de dados, campos obrigatorios, unicidade e workflow da OS estao definidos.
- As regras minimas de OS, estoque e faturamento estao registradas.
- Existe contrato conceitual de integracao com padrao de resposta e erros.
- Existe guia pratico de padroes, estrutura e governanca para desenvolvimento solo.
- Existem cenarios de validacao repetiveis e criterios de aceite por macrofluxo.
