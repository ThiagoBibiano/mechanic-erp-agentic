# mechanic-erp-agentic
ERP especializado para oficinas mecânicas de varejo construído sobre Frappe Framework. Integra gestão de Bounded Contexts (DDD) com uma camada agêntica inteligente via MCP para automação de serviços, estoque e financeiro.
